Final Project - Participatory Simulation



Scope of work

This project demonstrates using a participatory simulation to teach a machine learning algorithm. The project leverages the Q-Learning algorithm to teach an virtual agent to navigate an environment with walls.

The participatory simulation allows the student to "step into" the simulation to roleplay as the virtual agent.

This project is built completely using HTML and Javascript (there is no backend) to demonstrate how this project could work. The responses are saved in local storage to simulate the behavior if a backend were developed to store the responses.

For the best experience, feel free to try it as a student and submit responses and you will be able to see the responses as a teacher.
